스마트에디터에서는 단순히 글을 작성하는 기능 외에도 여러 첨부 기능과 글자 꾸미기 기능을 제공합니다.

**스마트에디터 2.0의 기능**  
**1. 각주 기능**  
글 작성 시 부연설명이 필요한 단어, 문장을 드래그한 후 각주를 입력하면, 등록 된 글을 읽을 때 각주 레이어 및 글 하단 각주 영역에서 해당 내용을 볼 수 있습니다.

**2. 표 편집 기능 개선**  
표를 삽입할 때 표 테두리 스타일을 선택할 수 있으며, 표 스타일을 16개 디자인 중에 선택할 수 있게 되었습니다.  
또한, 작성한 표를 드래그하면 간단 편집기를 통해 표구조 편집(삽입,삭제,분할,병합)등이 가능하고, 표 속성 편집(셀 배경색 및 스타일)을 수정할 수 있습니다.

**3\. 요약글 기능 개선**  
요약글을 1개 글 개수 제한 없이 작성할 수 있게 되었으며, 요약글내에 요약글을 5개까지 중첩하여 작성할 수 있습니다.

![](https://mblogthumb-phinf.pstatic.net/MjAxNzA0MjVfOTgg/MDAxNDkzMDg4Mjc4Njk0.2m1q6_B1nemVzkKgr9fqZ79nbxjSxmJA4gfOtnh1o6Yg.JK-QlUBKFh8qVccm3M_xflY8K0ZI_YXT_uFsTSim7kUg.JPEG.blogpeople/8-4_%EC%9A%94%EC%95%BD%EA%B8%80_%EA%B8%B0%EB%8A%A5_%EA%B0%9C%EC%84%A0.jpg?type=w800)

**4\. 글자효과 지우기**  
에디터에 글자 입력후 굵게, 밑줄, 글자색, 배경색 등의 글자효과를 넣었더라도 ‘글자효과지우기’를 통해 글자효과를 초기화할 수 있습니다.

![](https://mblogthumb-phinf.pstatic.net/MjAxNzA0MjVfMjAy/MDAxNDkzMDg4Mjk5NjEx.3C5uY3Co0m5kMbZAWS_IHEx-YU6Pw7BwquMQCsWWi90g.oNVRWdFAMdHMyUOwy_YawCTWPSe5QYr_5OmL5G6ZLS0g.JPEG.blogpeople/8-5_%EA%B8%80%EC%9E%90%ED%9A%A8%EA%B3%BC.jpg?type=w800)

**5\. 글자색, 글자 배경색**  
최근 사용 색상 버튼과 색상 팔레트 호출 버튼으로 구성되어, 최근에 사용된 색상이 보여지고, 바로 적용할 수 있습니다.

![](https://mblogthumb-phinf.pstatic.net/MjAxNzA0MjVfMzEg/MDAxNDkzMDg4MzEyODMy.fcQ4deFge4U9O3H23qOx_AG_dxmLSwulnTqQzA1Dq7Ag._F_zUX6iKIvqllcaJMdi5bip35AHVHfZzLBrX6QbgA8g.JPEG.blogpeople/8-6_%EA%B8%80%EC%9E%90%EC%83%89.jpg?type=w800)

**6. 줄간격 직접 입력 기능**  
기존에 제공되던 7단계의 줄간격 이외에 사용자가 직접 수치를 입력하에 줄간격을 미세하게 조절할 수 있습니다.

![](https://mblogthumb-phinf.pstatic.net/MjAxNzA0MjVfMTEw/MDAxNDkzMDg4MzM1MzE1.niVs0S5TCwSoOsV-Fz6lUJQoyrigD8SuAsV5hYC3on4g.JaCmtCTneEVrJoW0J6ixomQzmImqQoSPZUAgz_31_mMg.JPEG.blogpeople/8-7_%EC%A4%84%EA%B0%84%EA%B2%A9.jpg?type=w800)

**7\. HTML 직접 등록 방법**  
스마트에디터 기능 외에 HTML 소스를 직접 등록할 수 있습니다.  
다만, 제한된 HTML 소스 혹은 잘못 등록된 소스로 등록하실 경우에는 보안을 위한 필터링 혹은 오류가 발생할 수 있는 점 참고 부탁드립니다.
