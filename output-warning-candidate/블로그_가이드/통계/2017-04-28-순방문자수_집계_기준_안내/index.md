해당 기간 동안 내 블로그에 1회 이상 방문한, 중복되지 않은 방문자수를 의미합니다.  
동일인이라도 로그인 했을 경우와 로그인 하지 않았을 경우 각각 개별 산정됩니다.  
예를 들어, A가 로그인 상태로 아침에 방문하고, 로그아웃 상태에서 점심에 방문했다면, A는 순방문자수 2명으로 측정됩니다.
