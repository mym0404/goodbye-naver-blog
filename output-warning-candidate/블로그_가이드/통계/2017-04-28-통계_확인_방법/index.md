## Export Diagnostics

> ⚠️ Warning: SE3 블록을 해석하지 못해 건너뛰었습니다: se_component se_paragraph default

블로그 전체 통계는 PC와 모바일앱에서 확인 가능합니다.  
PC는 내 블로그 관리 > 상단의 \[통계\] 메뉴를 클릭해주세요.  
모바일앱은 표지에서 확인 가능합니다.
