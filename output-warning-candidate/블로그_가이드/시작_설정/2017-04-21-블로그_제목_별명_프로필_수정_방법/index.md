블로그 제목과 별명은 블로그의 주제와 나를 표현하는 기본적인 부분입니다.

**제목, 별명, 소개글, 프로필 사진 변경 방법**  
변경 페이지 접속은 2가지 경로로 제공되고 있습니다.  
**1\. 내 블로그 > 프로필 > edit**  
**2\. 내 블로그 > 관리 > 기본 설정 > 블로그 정보**

1\. **제목** : 블로그 타이틀이 생성되어 있을 때 노출이 됩니다.  
2\. **별명** : 포스트 작성 시에 노출되지 않고 댓글 작성 시에만 노출됩니다.  
 별명을 설정하지 않은 경우 댓글을 등록하면 아이디가 노출되고,  
 별명을 설정한 경우에는 댓글 등록 시 별명이 노출됩니다.  
3\. **소개글** : 블로그 프로필 영역에 노출됩니다.  
4. **블로그 프로필 사진** : 댓글 작성시 "프로필형"으로 설정된 블로그에서 노출됩니다.  
**※ 참고해주세요!**  
\-변경한 제목과 별명은 일정한 시간 후 지나야만 검색 결과에 반영됩니다.
