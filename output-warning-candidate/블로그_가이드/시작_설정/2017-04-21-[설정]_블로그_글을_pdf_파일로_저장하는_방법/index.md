## Export Diagnostics

> ⚠️ Warning: SE3 블록을 해석하지 못해 건너뛰었습니다: se_component se_paragraph default

블로그에 등록한 글을 PDF 파일로 백업하는 기능을 제공하고 있습니다.  
\[글 저장\] 기능을 이용하여 직접 등록한 게시글을 간직해 보세요.  
****글 저장하는 방법****  
**1. \[블로그관리> 글관리> 글 저장\] 을 클릭합니다.**  
**2. 저장할 파일제목을 입력하고 기간/카테고리/공개설정에 따라 글 목록을 불러 옵니다.**

▲ 글 저장 ( PDF 파일 제목 )

**3. 목록을 살펴보고 저장을 원하는 글 제목 앞 체크박스에 체크를 합니다.**  
 전체선택을 원하신다면 좌측 상단의 체크박스를 이용해 주세요.  
**4. '추가하기' 버튼을 클릭해주세요. 아래쪽 박스에 선택한 글 목록이 만들어집니다.**  
**5. 상단 좌측의 순서조정 버튼과 체크박스를 이용하여**  
 **전체삭제와 개별삭제, 개별 글의 순서바꾸기가 가능합니다.**  
 글 정렬순서를 최근 작성일순과 나중 작성일순으로 선택 할 수 있습니다.

▲ 글 저장 ( 글 선택 )

**6. 저장 할 글 선택이 끝나셨다면 파일제목을 입력하셨는지 꼭 확인하시고**  
 **'만들기' 버튼을 클릭해주세요.**  
**7. 저장목록 탭에서 PDF 파일이 생성 중인 것을 확인할 수 있으며**  
 **상태가 '다운로드'로 바뀌면 생성이 완료된 것입니다.**  
**8. 제작된 파일의 수정은 불가하며 삭제 후 새롭게 제작하실 수 있습니다.**  
※ 참고사항  
\- 본인 작성 블로그 글만 저장 지원 (메모, 스크랩, 징계 글 제외)  
\- 폰트는 나눔고딕으로 지원. 본문 폰트 사이즈 10P로 동일하게 제공됨  
\- 글 정렬은 좌측 정렬만 지원  
\- 글양식, 글장식 등의 레이아웃은 원문형태로 지원하나  
 PDF 파일에서의 호환성 문제로 좌우여백의 불균형이 일부 있을 수 있음  
\- 스토리포토, 플래시, 뮤직플레이어, 파일, 투표, 일정 등의  
 첨부양식은 표시되지 않으며 지도와 동영상은 썸네일로 표시됨  
\- 원본사이즈가 큰 이미지의 경우, PDF 글의 사이즈에 맞추어 조정되어 보여짐  
 (웹과 동일)  
\- 저장된 글은 다시 블로그로 불러올 수 없으며, PDF로 저장되어 보여지는 기능임  
※ 지원 용량  
\- PDF당 허용 글 개수(100개)  
\- PDF당 허용 용량(500MB)  
\- 전체 허용 용량(3GB) = 1일 최대 생성 용량과 동일함  
\- 생성가능 PDF 개수(20개)
