## Export Diagnostics

> ⚠️ Warning: SE3 블록을 해석하지 못해 건너뛰었습니다: se_component se_paragraph default

댓글과 답글 입력 시, 내용 입력란 오른쪽에 '주인만 보기'와 '비밀 답글' 메뉴가 제공되며 박스에 체크하여 사용할 수 있습니다.

**비밀 댓글/답글 작성 방법**  
답글 작성 시 "비밀 답글"에 체크하면 댓글을 작성한 상대방에게만 노출됩니다.
