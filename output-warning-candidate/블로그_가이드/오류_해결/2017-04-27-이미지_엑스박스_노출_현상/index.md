등록한 사진이 엑스박스로 노출되는 원인은 세가지가 있을 수 있습니다.

**1\. 썸네일이 생성되지 않는 경우**  
**※ 썸네일이란?**  
포토업로더로 사진 등록 시 생성되는 작은 이미지  
포토업로더로 등록한 사진은 하단에 이미지 썸네일이 생성이 되며,  
이미지 썸네일이 생성되지 않으면 일정 시간 이후에 엑스박스로 노출됩니다.  
**1) 썸네일 확인 방법**

▲ 스마트에디터 2.0 썸네일 위치

▲ 스마트에디터 3.0 썸네일 위치

사용하시는 스마트에디터 버전에 따라 썸네일이 위치하는 곳이 다르며,  
생성 여부를 확인해 주시면 됩니다.  
지속적으로 썸네일이 생성되지 않는다면 아래의 도움말을 참고하여 브라우저 최적화를 해주세요.  
**[이미지/동영상 등록 및 썸네일 생성이 되지 않아요.](https://help.naver.com/support/contents/contents.nhn?serviceNo=520&categoryNo=10036)**  
포토업로더로 등록한 이미지의 썸네일이 생성되지 않는 현상의 경우  
**맨 아래의 항목을 선택해 주시어 문의해 주시면 복구 가능 여부를 확인하여**  
**이미지 복구를 도와 드리겠습니다.**

**2\. 복사/붙여넣기로 등록한 이미지인 경우**  
다른 게시글 혹은 블로그나 사이트의 이미지를 복사/붙여넣은 경우,  
원본 사진이 삭제되거나 이동, 차단 등의 현상이 발생하면 엑스박스로 노출됩니다.  
**복사/붙여넣기로 등록된 이미지는 복구가 안 되며 게시글마다 포토업로더를**  
**이용하여** **등록해 주셔야 정상적으로 노출이 가능하오니 다시 등록해 주세요.**  
등록 시 동일하게 복사/붙여넣기로 등록하면 동일 현상이 발생하오니  
꼭 포토업로더를 이용하여 새로 등록해 주셔야 합니다.

**3\. 사용하시는 브라우저에서 이미지를 읽어오지 못하는 경우**  
썸네일은 정상적으로 있지만. 특정 환경의 브라우저 혹은 모바일에서만  
이미지가 나오지 않는다면 아래와 같이 브라우저 최적화를 해주세요.  
**[원클릭 진단](https://help.naver.com/support/service/main.nhn?serviceNo=520&categoryNo=10166)**  
또한, 네트워크 환경을 변경하여 보시면 정상 노출 여부가 확인될 수 있으니  
참고 부탁드립니다.
