## Export Diagnostics

> ⚠️ Warning: SE3 블록을 해석하지 못해 건너뛰었습니다: se_component se_paragraph default

모바일 블로그 사용 중 발생하는 오류 현상으로 불편을 겪으시는 고객님께 조금이라도 도움을 드리기 위해 많은 분들이 "이거 하나로 해결되었어요."라고 말씀해 주신 방법을 안내해 드립니다.  
**1\. 실행 중인 앱 종료**  
홈 버튼을 두 번 누르면 실행 중인 앱들이 화면에 노출되며,  
종료할 앱을 누른 상태로 위로 올리면(플리킹) 실행 중인 앱이 종료됩니다.

![](https://mblogthumb-phinf.pstatic.net/MjAxNzA0MjdfMTAx/MDAxNDkzMjcxODM3NTE3.HzzZFIHVhQkNcHgpFwpYIt9olURsxwveISOvcWAxnLIg.IpsWByz-2CkipUIAwiwyb5qkcOnsvZRoWILcFPGXnksg.JPEG.blogpeople/%EC%A0%84%ED%99%94.jpg?type=w800)

**2\. 웹 캐시 삭제**  
**\[설정\] >\[Safari\] > \[쿠키 및 데이터 지우기\] > \[쿠키 및 데이터 지우기\]**  
**※ 단, 캐시 및 데이터 삭제 시 이전에 사파리로 사용하셨던 기록들이 삭제될 수 있는 점 참고 부탁드립니다.**

![](https://mblogthumb-phinf.pstatic.net/MjAxNzA0MjdfMjA3/MDAxNDkzMjcxOTQzNTI3.iq14fXOqB2-S7T-mCtRzx9dJ3GGA3feSiQZ4tT42dIwg.ScHB5kaY_qYNs4-V-iSlraHsvWLJG0Y9F7ik5Wbs8tcg.JPEG.blogpeople/%EC%95%84%EC%9D%B4%ED%8F%B0_%EC%84%A4%EC%A0%95.jpg?type=w800)

****3\. 날짜와 시간 설정**  
******\[설정\] > \[일반\] > \[날짜와 시간\] > \[자동\]**으로 설정\- 이미 자동으로 체크가 되어 있다면 체크 해제 후 다시 체크하여 재부팅까지 진행해 주세요.

![](https://mblogthumb-phinf.pstatic.net/MjAxNzA0MjdfMjgx/MDAxNDkzMjcxOTg2NDU1.kW_3VzL1thZsVAAnpU33fHYbwvsNQI0y6dcm8VLEvJUg.ZsQxnDWBGFHWQGQEWefKKpodZpSjLAabrW6ReN9VVkUg.JPEG.blogpeople/%EC%84%A4%EC%A0%95_%EC%9D%BC%EB%B0%98.jpg?type=w800)

**4\. 앱 삭제 후 재설치**  
모바일 앱을 이용하면서 오류가 발생했다면 앱을 재설치하는 방법을 권해 드립니다.  
앱을 삭제하신 후 모바일 기기를 종료 후 전원을 다시 켜주세요.  
**※ 삭제하실 앱을 꾹 누르고 있으면 아래 화면과 같이 노출되며 x 버튼 누르면 해당 앱이 삭제됩니다.**  
다만, 앱 삭제 후 재부팅시에는 휴대기기에 저장된 임시저장된 게시글이 삭제되오니 필요하신 임시저장 게시글은 PC로 전송해서 백업해주세요.  
**※ \[블로그앱\] > \[글쓰기\] > \[임시저장\] > \[PC 전송\]**
