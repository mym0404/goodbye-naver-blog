요약  
1\. 스마트에디터 3.0에서 본문 공유가 지원되지 않는 부분을 가장 불편하다 말씀해주셨습니다.  
기억나는 댓글들도 정말 많아요. ㅠㅠ 여러분이 불편하다 하시면 저희는 바꾸겠습니다.  
올가을을 기대해주세요. 코피 나도록 개발해보겠습니다.  
2\. 포토 업로더에서는 1장당 첨부 용량 제약을 가장 불편하다 말씀해주셨습니다.  
이미지 1장당 20M로 용량 제한을 늘리려고 합니다. 7월에 바로 개선하겠습니다.  
뿐만아니라 가벼워진 포토 업로더도 6월 말 서비스 적용을 목표로 개발 진행 중입니다.  
움짤도 할 수 있고, 탐색기에서 드래그해 이미지를 첨부하실 수도 있습니다.  
**6월 말, 기대해주세요!**

#모바일 내 블로그홈에 대한 댓글의견

가장 많았던 의견이어서, 서비스 적용이 시급하다고 생각되는 기능

![스티커 이미지](https://storep-phinf.pstatic.net/blogc/original_9.png?type=p100_100)

> 프로필 사진 누르면 사진이 크게 보이면 좋겠습니다.  
> 인기글은 좋지만, 공지사항이 눈에 띄지 않는 것은 아쉽습니다.  
> 외부채널 블록이 PC에도 보이면 좋겠습니다.  
> 블로그 주인이 먼저 보여주고 싶은 글이나 카테고리가 눈에 잘 띄면 좋겠습니다.  
> 강조하고 싶은 카테고리를 인기글처럼 설정하고 싶습니다.  
> 내 블로그 홈의 각 요소들의 순서를 내 맘대로 조정하고 싶습니다.  
> 페이스북 페이지를 등록하고 싶습니다.

서비스 적용에 어려움이 있는 부분들도 있습니다.

![스티커 이미지](https://storep-phinf.pstatic.net/blogc/original_15.png?type=p100_100)

> SNS 계정을 여러 개 등록하고 싶습니다.  
> 페이스북의 주소가 깔끔한 형식으로 표현되면 좋겠습니다.

현재는 SNS 연동은 네이버 ID 당 1개씩만 연동되도록 관리하고 있어서,  
하나의 SNS 서비스에 여러 개의 계정을 입력하기가 (현재로서는) 기술적인 검토가 더 필요한 상황입니다.  
페이스북의 주소가 더 깔끔한 형식으로 표현되면 좋겠다 말씀해주신 부분은  
조금 더 면밀한 개발 검토가 필요하여, 당장은 해결하기가 어려울 것 같습니다. ㅠㅠ
