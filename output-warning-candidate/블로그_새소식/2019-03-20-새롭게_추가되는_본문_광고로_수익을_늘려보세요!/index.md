안녕하세요, 여러분! 네이버 블로그팀입니다.

이제 날씨가 점점 따뜻해져서 진짜로 봄이 온 것 같습니다.

그래서! 오늘은 봄처럼 두근두근 설레는 소식을 하나 전달드리려고 합니다!

그것은 바로! 블로그 글로 수익을 얻을 수 있는 방법이 하나 더 생겼다는 소식인데요!

3월 27일부터는 기존에 사용 가능했던 글 하단 광고(파워링크) 뿐만 아니라,

블로그 글 본문에서도! 광고를 노출하실 수 있게 된답니다!

광고 영역이 하나 더 늘어났으니, 수익도 그만큼 쑥~쑥! 올라가겠죠?

본문에 광고를 넣게 되면서 블로그에 좋은 글을 써주시는 여러분들이

이전보다 더 많은 수익을 가져가실 수 있게 되길 진심으로 바랍니다.

그럼, 지금부터 새롭게 선보이는 본문광고에 대해 자세히 설명드릴게요~!

> **노출 위치**

본문광고는 블로그 글의 마지막 부분인 태그 영역 바로 위에, 아래 보여드리는 3가지 유형이 랜덤으로 보여집니다.

사전 테스트를 통해 여러분의 수익이 가장 많이 나올 수 있는 최적의 위치와 디자인을 고민해서 적용했습니다!

> **적용 방법**

이미 애드포스트에 가입하신 분들은 3월 27일 기존 작성한 글에 본문광고가 자동으로 적용되고, 새 글은 발행하는 동시에 광고가 적용됩니다.

아직 애드포스트에 가입하지 않으셨다면, 지금 바로 애드포스트 사이트에 가입해주세요. 간단한 심사가 끝나면 본문광고가 자동으로 적용된답니다~! →[애드포스트 가입하기](https://adpost.naver.com/)

**Q&A**

블로거분들이 궁금해하실 질문에 미리 답해드립니다!

Q : 본문광고는 전체 문서에 모두 노출되나요?

A : 아니요, 본문광고는 스마트에디터2.0로 작성된 글과 스크랩한 글에는 노출되지 않습니다.

Q : 기존 하단광고(파워링크)는 그대로 노출하고, 본문광고는 노출하고 싶지 않습니다.

A : 애드포스트 설정페이지에서 본문광고 사용 설정만 '사용하지 않음'으로 선택해주시면 본문광고는 노출되지 않습니다. 하단광고(파워링크)만 별도로 제어할 수 있는 기능은 제공되지 않습니다. 양해 부탁드립니다.

Q : 기존 애드포스트 위젯은 이제 사용할 수 없나요?

A : 네, 앞으로는 애드포스트 설정페이지에서 진행하시면 됩니다. 훨씬 편하실 거에요!

Q : 본문광고 ON/OFF 설정은 PC에서만 가능한가요?

A : 모바일앱에도 4월 8일 적용될 예정입니다. 조금만 기다려주세요!

\*\* 모바일앱 적용이 4월 15일로 변경되었습니다. 좀더 꼼꼼히 검토하여 안정적인 기능을 제공해드리기 위함이니, 양해 부탁드립니다. (\_ \_)

Q : 본문광고 위치는 자유롭게 설정할 순 없나요?

A : 네, 현재는 블로거분들과 이용자분들께 안정적인 광고를 제공하고 관리하기 위해 제한하고 있습니다만, 향후 필요에 따라 규정이나 기능 등을 재검토할 계획입니다.
