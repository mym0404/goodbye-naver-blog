안녕하세요. 블로그팀입니다.

모바일 블로그 하단 애드포스트 영역 개선이 예정되어 안내드립니다.

---

**1\. 광고 위치 변경**

먼저, 모바일 애드포스트(파워링크) 위치는 기존 맨 하단에서 프로필 하단으로 변경됩니다. 지난해 테스트를 통해 광고노출 위치의 변화가 개별 블로거분들에게 돌아가는 수익변화에 영향을 끼치는 결과를 확인하였습니다. 테스트 결과를 바탕으로 광고 소재를 프로필 하단으로 이동하게 되었습니다.

**2\. 광고 UI 개선**

더불어, 기존 텍스트로만 구성되있던 광고 소재에 이미지가 추가됩니다. 블로그의 콘텐츠 목록과 더욱 어울리는 이미지형이 추가됨으로써 기존 텍스트형보다 더 깔끔하고 보기 좋게 개선하였습니다. 아래 예시 이미지를 보시면 더 이해가 빠르실거에요!!

※ 애드포스트 설정하신 분들은 아래 내용 꼭 확인해주시고 블로그 운영시 참고해주세요.
