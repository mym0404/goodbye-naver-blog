한눈에 볼 수 있는 일간현황!

> 통계 데이터 강화

그리고 더욱 강화된 통계 데이터들!  
**유입분석 강화** - 검색 유입분석과 사이트 유입분석을 구분하여 어떠한 글이 검색을 통해 더 많이 보여졌는지 확인이 가능하도록 변경하였습니다. 또한 메인 유입경로를 기존 10개에서 20개까지 확대하여 최대 2100개의 메인 유입경로를 볼 수 있도록 변경하였습니다.  
**실시간 데이터 추가 제공** - 실시간으로 추가된 이웃수와 방문자의 성별, 연령별 분포를 추가로 제공하고, 오늘의 조회수 순위를 최대 10개에서 30개까지 확대 적용하여 발행 글 수가 많고, 유입자수가 많은 블로거들도 편리하게 이용할 수 있도록 변경하였습니다.  
**게시글별 지표 강화** - 어제와 지난주만 조회 가능했던 게시글별 지표는 일자별 조회 기능을 추가하고, 일별, 주별, 월별 기간선택 기능을 추가로 신설하여 선택 게시글이 어떠한 흐름으로 반응을 받고 있는지 확인 할 수 있도록 변경하였습니다. 각 게시글별 최대 26개월의 데이터를 조회할 수 있는데요, 많은 사용자들이 기능 확대를 원했던 부분이기도 하지요!  
**성별, 연령별 분포** - 기존에는 주간/월간 지표만 제공하던 것을 일간 지표를 추가하여 내 게시글이 어떠한 성별에 더 반응하고 있는가를 빠르게 볼 수 있도록 하였습니다.

> 모바일 화면 개선

좁은 모바일 화면에서 더욱 편리하게!  
각 지표별 바로가기 메뉴 이동 버튼을 제공하여 페이지 이동을 최소화 할 수 있도록 하였습니다. 상세 메뉴역시 스크롤 다운 방식에서 탭방식으로 변경하여 좁은 모바일 화면에서도 불편함이 없도록 개선하였습니다.
