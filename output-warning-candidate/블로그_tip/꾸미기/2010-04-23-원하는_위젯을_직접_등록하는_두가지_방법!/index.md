위젯 직접 등록 기능을 활용하시면, 네이버에서 기본으로 제공해드리고 있는 위젯 외에도 다양한 기능과 디자인의 웹 위젯으로 내 블로그를 꾸밀 수 있답니다!

그럼, 마음에 드는 위젯을 직접 등록하는 방법을 알려드릴게요.

**\[방법1\] 위젯 코드 직접 입력하기**

직접 제작해둔 위젯이 있다면, 위젯 코드를 내 블로그의 '위젯직접등록' 팝업에 붙여넣기만 하면 간편하게 등록 완료!

![](https://mblogthumb-phinf.pstatic.net/20160615_241/blogpeople_1465966723285ffEm4_PNG/%C0%A7%C1%AC%C1%F7%C1%A2%B5%EE%B7%CF%C3%A2.PNG?type=w800)

**\[방법2\] 텍스트 또는 간단한 HTML 편집으로 나만의 위젯 만들기!**

내 블로그의 '위젯직접등록' 팝업에 텍스트를 입력하는 것만으로도 나만의 공지 위젯을 만들 수 있습니다.

간단한 HTML 편집을 하실 수 있다면, HTML을 직접 입력하셔서 나만의 배너나 예쁜 메모 위젯도 달 수 있어요.

[☞ 메모 알림판 위젯 만들기 예제](http://blog.naver.com/blogpeople/150085821418)

[☞ 공지사항 바로가기 위젯 만들기 예제](http://blog.naver.com/blogpeople/150085821382)

두 가지 방법을 활용해서

나만의 블로그를 더 잘 꾸밀 수 있는 위젯을 등록해보세요.

블로그팀 드림.
