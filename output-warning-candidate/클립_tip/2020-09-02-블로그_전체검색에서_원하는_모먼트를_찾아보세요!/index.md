블로그 모먼트 업데이트 소식도 손에 들고 왔습니닷.

이번 **블로그앱 업데이트**를 하시면,

**블로그 내 검색에서 모먼트를 만나보실 수 있습니다.******

네이버 통합검색에서는 글과 함께 보여서, 모먼트만 쏙쏙 - 찾기 어려웠는데

블로그 전체검색에서는 검색어로 모먼트 동영상만 뙇! 찾으실 수 있어요.

화면을 보면서 설명 드릴게요~

![](https://mblogthumb-phinf.pstatic.net/MjAyMDA4MjdfMTI5/MDAxNTk4NDkyNTkyNDY0.c_HDlnoMtlp35gVKEbIaQHiqMJLg6dP-YM1FDzx6B24g.J31xe8eK5VcUOQSXMYjGz6GIKMjFQbiqw-uJHaKfhTYg.PNG.blogpeople/%EB%AA%A8%EB%A8%BC%ED%8A%B8%ED%99%88_%EC%98%A4%ED%94%88%EA%B3%B5%EC%A7%80_%EB%AA%A8%EB%A8%BC%ED%8A%B8%EA%B2%80%EC%83%89_01.png?type=w800)

모바일 블로그 이웃새글

또는 모먼트 홈에서

\[돋보기\] 검색 버튼을 터치하고 검색어를 입력하면

글 / 동영상 / 모먼트 / 태그 / 사용자

총 5가지 기준으로 검색결과를 확인하실 수 있습니다.

\+ 앗!

**모먼트 홈 상단에 6가지 주제 버튼도 추가되었다는 사실~**

여행,맛집 /일상 /레시피 /리빙 / 패션뷰티 /상품리뷰

관심있는 주제를 터치하시면, 모먼트를 주제별로 모아볼 수 있는 페이지로 이동합니다.

**블로그 전체 검색**
