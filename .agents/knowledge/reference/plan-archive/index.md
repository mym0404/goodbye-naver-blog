# Plan Archive

## 목적
완료된 플랜 실행 기록과 큰 변경의 검증 증거를 보관하는 인덱스다. 진행 중 메모는 여기 두지 않고, 닫힌 작업만 남긴다.

## 포함 기준
- 구현과 검증이 끝난 큰 변경
- 이후 변경에서 다시 참고할 가치가 있는 실행 계획
- 결과 판단에 직접 연결되는 evidence 묶음

## 현재 아카이브
- PicGo upload progress visibility: `.agents/knowledge/reference/plan-archive/picgo-upload-progress-visibility/plan.md`

## 운영 규칙
- 새 큰 변경을 닫을 때는 `plan.md`, `tasks.md`, 필요한 `notes.md`, `evidence/`를 같은 번들 폴더에 보관한다.
- 진행 중 scratch나 임시 메모는 완료 전까지 다른 작업 공간에 두고, 닫힌 뒤 필요한 기록만 여기로 옮긴다.
- 제품 계약으로 승격된 내용은 상위 evergreen 문서에 반영하고, 여기에는 실행 기록과 증거를 남긴다.
